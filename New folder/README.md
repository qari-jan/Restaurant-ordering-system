
Overview
A Python **Object-Oriented Programming (OOP)** project that simulates a restaurant ordering and billing system.  
It allows customers to order food items, split bills, and manage menu dynamically using text files.

Features
- Dynamic menu loading from `menu.txt`
- Case-insensitive user input
- Bill splitting among multiple people
- Orders and bills saved in `bill.txt`
- Modular OOP design with classes:
  - Food_Item
  - Order
  - Person
  - Bill_Splitter
